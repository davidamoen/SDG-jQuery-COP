Loading jQuery

Either download the latest file from here and add this into a script tag

or

Just add this script tag: 

<script language="javascript" type="text/javascript" src="http://code.jquery.com/jquery-1.7.2.min.js"></script>

Then start a custom.js file with this: 

$(document).ready(function () {
    // jQuery code goes here
});

Traversing the DOM

Get a collection of all input tags on a page and do something

    $(input).attr('disabled','disabled');
    $(a).last().click();

Use CSS IDs and classes to identify elements

    $('#myHelpText').show();
    $('#myForm').submit();

Use CSS classes to identify groups of elements

    $(a.active).attr('href','http://www.solutiondesign.com');
    $('.inactive').css('color', 'gray');

Be more specific

    $('.wrapper .right a).addClass('foo');

Additional methods

    $('.content').children().show();
    $('.active').prev().hide();
